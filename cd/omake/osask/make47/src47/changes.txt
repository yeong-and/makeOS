·화면 모드 변환이 실패했을 경우에 원래대로 되돌리는 기능을 수정했습니다.

Version 3.3(2003/2)

·ankfont0를 버전 업 하여 聖人씨의 그래픽 캐릭터를 도입했습니다.

Version 3.2(2003/1)
[OSASK 5548][OSASK 5557]
·C의 원시 파일의 컴파일러가 GO(cc1 + as)에서 GO(cc1 + gas2nask + nask)로 변경되었습니다.
·mousespeed의 디폴트가 2가 되었습니다.
·신규 파일 생성을 위한 API가 정비되었습니다.
·압축해도 1MB를 넘는 어플리케이션의 전개에 대응했습니다.
·聖人씨의 일자 표시 개조를 수중에 넣었습니다(PC-AT판만).
·압축파일의 resize0를 허용 했습니다.
·어플리케이션의 IOPL를 3으로 변경했습니다.

Version 3.1(2002/12)
[OSASK 5364][OSASK 5365][OSASK 5405]
·자기 시그널 송신 API의 버그가 수정되었습니다.
·깜박 잊고 있었던 디버그 메세지를 삭제했습니다.
·PC/AT판에 대해 키보드 인터페이스 주위가 변경(초기화시의 ACK/NACK 이외
  의 응답을 무시) 되어 대응하는 키보드가 증가했습니다.
·C의 원시 파일의 컴파일러가 lcc로부터 GO(cc1 + as)로 변경되었습니다.
  이것에 의해 tolset02를 다운로드하면 컴파일에 필요한 툴을 모두
  갖출 수가 있습니다.

Version 3.0(2002/11)
[OSASK 5137][OSASK 5142][OSASK 5143][OSASK 5183]
·각 모듈을 미리 압축해 두는 것으로, OSASK.EXE가 작아졌습니다.
·부트 순서가 변경되어 실 메모리의 사용량이 적어졌습니다.
·PC/AT판에 대해 키보드 인터페이스 주위가 변경되었습니다.
·기동시에 OSASK.EXE에 부속된 디스크 이미지를 전개하는 것으로써,
  플로피 디스크가 없어도 기동할 수 있게 되었습니다.
·TOWNS판에서의 직접 기동할 수 있는 디스크 생성이 서포트되었습니다.
·빌드시의 링커가 obj2bim3 + bim2bin3 + osalink1에 버전 업 되었습니다.

Version 2.9(2002/10)
·빌드 타겟으로서 nec98가 추가되었습니다.
　현상은 특정의 기종으로 동작 체크를 행한 정도이며, 동작 확인은 충분하다고는 
할 수 없습니다(주：NEC98상에서 빌드 할 수 있을지 어떨지는 화제가 되고 있음).
　이것에 따라 nec98 디렉토리가 추가되었습니다
·빌드시의 링커가 obj2bim2 + bim2bin2 에 버전 업 되었습니다.
·표준 라이브러리가 stdlib6에 버전 업 되었습니다.
